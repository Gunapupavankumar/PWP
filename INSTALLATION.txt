========================================
HEALTHCARE WELLNESS PORTAL
Quick Installation Guide
========================================

PROBLEM: PowerShell Script Execution Error
SOLUTION: Use Command Prompt (CMD) instead

========================================
STEP-BY-STEP INSTALLATION
========================================

1. Open Command Prompt (CMD) - NOT PowerShell
   - Press Windows Key + R
   - Type: cmd
   - Press Enter

2. Navigate to project folder:
   cd C:\Users\nikhi\OneDrive\Desktop\HCL_TECH\healthcare-portal

3. Install dependencies:
   npm install

4. Start JSON Server (Terminal 1):
   npm run server
   
   Keep this terminal open!

5. Start React App (Terminal 2):
   Open a NEW Command Prompt window
   cd C:\Users\nikhi\OneDrive\Desktop\HCL_TECH\healthcare-portal
   npm start

========================================
ALTERNATIVE: Use Batch Files
========================================

Double-click these files in order:
1. install.cmd (installs dependencies)
2. start-server.cmd (starts JSON server)
3. start-app.cmd (starts React app)

========================================
ACCESS THE APPLICATION
========================================

React App: http://localhost:3000
JSON Server: http://localhost:5000

========================================
TROUBLESHOOTING
========================================

If npm commands don't work:
1. Make sure Node.js is installed
2. Use CMD, not PowerShell
3. Run as Administrator if needed

If port 3000 or 5000 is busy:
- Close other applications using these ports
- Or change ports in package.json
